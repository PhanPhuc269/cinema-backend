# Cinema Booking Service
This microservice is responsible for:
* User onboarding
* Managing roles and permissions
* Booking tickets and managing showtimes

## Tech stack
* Build tool: maven >= 3.9.5
* Java: 21
* Framework: Spring boot 3.2.x
* DBMS: MySQL

## Prerequisites
* Java SDK 21
* A MySQL server

## Start application
`mvn spring-boot:run`

## Build application
`mvn clean package`
